# 不可替代的你.skill

> **irreplaceable-you** — 蒸馏你的不可替代性，让任何AI真正理解你是谁。

## 核心理念

真正让你不可替代的，不是你知道什么，而是你做事的方式。
真正难以被AI替代的，不是技能，是你作为一个独特个体看待世界的角度——而这个角度，是可以被提炼的。

## 核心能力

- **发现你的核心信念、决策哲学、问题意识**
- **提取你的表达纹理和价值观层级**
- **生成不可替代性档案**（identity.md）
- **生成AI Persona**（persona.md）——让AI真正用你的声音思考，而不是通用地回答问题

## 安装

```bash
git clone https://github.com/xiaoshiyilangzhao1996-droid/irreplaceable-you.git \
  ~/.openclaw/workspace/skills/irreplaceable-you
```

## 使用

```
/irreplaceable-you
```

或直接说：**创建我的不可替代性 Skill / 蒸馏我自己 / 我是谁 Skill**

## 触发词

创建我的不可替代性 · 蒸馏我自己 · 我是谁 Skill · 建一个我的镜像 · 追加新信息 · 更新我的不可替代性

## 三种工作模式

**新建模式**：从头构建你的不可替代性 Skill
**进化追加**：补充新的对话、文件或口述内容
**进化纠正**：修正已有的描述偏差
