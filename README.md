# 不可替代的你.skill

蒸馏你的不可替代性，让任何AI真正理解你是谁。

## 核心理念

真正让你不可替代的，不是你知道什么，而是你做事的方式。

## 功能

- 发现你的核心信念、决策哲学、问题意识
- 提取你的表达纹理和价值观层级
- 生成不可替代性档案（identity.md）
- 生成AI Persona（persona.md）

## 安装

```bash
git clone https://github.com/your-repo/irreplaceable-you ~/.openclaw/workspace/skills/irreplaceable-you
```

## 使用

```
/irreplaceable-you
```
